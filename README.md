# GraphPoolingGarden
A repo for baseline of graph pooling methods.

[中文](/README_CN.md)

## Datasets
- TuDataset
  - D&D
  - PROTEINS
  - ENZYMES
  - NCI1/NCI109
  - Reddit-Binary
- OGB
  - ogbg-molhiv
  - ogbg-ppa
  - ogbg-code2

## Pooling methods
- set2set
- sagpool(sequence/hierarchical)
- graph-U-net

### Model Framework

-hierarchical model
-sequencial model
-U-net-like model

## Readout methods
- mean
- max
- sum
